Django
1.什么是Django
    Django是一个开源框架,2005年发布,采用python语言编写的
    早起主要是做新闻和内容管理
    Django本身提供了非常强大的后台管理系统

    版本: 2.1.3
    使用: 1.11.8

   官网:http://www.djangoproject.com

2.框架模式
    MTV
        M: Models
        T: Templates
        V: Views

3.Django的安装
    1.查看自己安装的Django版本
        1.进入到终端以及python交互模式
            交互模式 python3 / ipython3
        2.交互模式中输入: import django
            1.如果未报错,:当前环境中已经安装django
            2.如果报错: 没有django
        3.查看版本:
            交互模式中:django.VERSION

    2.安装Django
        1.在线安装
            pip3: 安装到python3下
            pip2: 安装到python2下

            sudo pip3 install django(默认安装最新版本)

            sudo pip3 install django==1.11.8
            (指定安装Django的1.11.8)

        2.离线安装
            1.下载安装包
            2.sudo pip3 install 包名
